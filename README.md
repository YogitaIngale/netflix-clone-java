# Netflix Clone - Spring Boot Project

## Project Overview
Netflix Clone is a full-stack web application inspired by the Netflix streaming platform. The project is developed using Java Spring Boot for the backend 
and modern frontend technologies for the user interface. It provides a responsive and interactive movie streaming platform design with features like movie browsing, 
categorized content sections, user-friendly navigation, and backend integration.

The project focuses on implementing full-stack development concepts including frontend design, backend APIs, database connectivity, and MVC architecture.

---

## Features
- Responsive Netflix-inspired UI
- User-friendly homepage design
- Movie and TV show categories
- Dynamic content rendering
- Backend integration using Spring Boot
- Database connectivity with MySQL
- REST API support
- Navigation bar with interactive UI
- Responsive design for desktop and mobile devices
